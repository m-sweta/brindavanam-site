BRINDAVANAM WEBSITE
===================
This folder is your whole website.

  index.html   <- the page. ALL text, headlines, phone, email live here.
  images/      <- every photo, one .jpg per file. Swap a photo by replacing
                  the file with the same name (keep the name the same).

TO EDIT TEXT:
  Open index.html, use Find (Ctrl/Cmd+F) to locate the words you want to
  change, type the new words, save. On GitHub: click the pencil icon, edit,
  Commit. Netlify republishes automatically in ~30 seconds.

TO REPLACE A PHOTO:
  Put your new image in images/ using the EXACT same filename (e.g.
  nb_hero.jpg). Keep it a .jpg. That's it.

TO ADD YOUR CONTACT INFO:
  Search index.html for: (000) 000-0000   and   hello@brindavanam.com
  Replace both with your real phone and email (they appear in two places).
